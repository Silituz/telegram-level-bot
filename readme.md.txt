# Telegram Level-Bot

Ein Telegram-Bot, der XP, Level und Haustiere vergibt – perfekt für Gruppen! 🐍🐺🐱🐶

## Features
- XP-System mit Levelaufstieg
- Haustiere sammeln und benennen
- Täglicher XP-Bonus
- Flask-Webserver für keep-alive (z. B. Render)

## Start
```bash
pip install -r requirements.txt
python main.py
